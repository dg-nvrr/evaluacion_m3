# Gestor de Tareas como evaluacion final de modulo - dg-nvrr

Aplicación de línea de comandos en Python para organizar pendientes diarios. Permite crear, listar, completar y eliminar tareas desde la terminal.

## Características
* **Interfaz Simple:** Menú numérico fácil de usar.
* **Persistencia de Datos:** Las tareas se guardan automáticamente en un archivo `tareas.json`, por lo que no se pierden al cerrar el programa.
* **Manejo de Errores:** Validaciones para evitar fallos al ingresar opciones incorrectas.

## Requisitos
* Python

## Cómo ejecutar
1.  Clona este repositorio o descarga los archivos.
2.  Abre la terminal en la carpeta del proyecto.
3.  Ejecuta el siguiente comando:

```bash
python gestor_tareas.py
